Тестовое задание по Андроиду


Написать приложение, которое показывает список мест с сервиса Google Places API Web Service.

  1. Показывать список мест в окрестностях текущей позиции. (Поиск мест поблизости)

  2. Реализовать возможность просмотра полученных мест на карте.

  3. Для каждого элемента отображать название и дистанцию до него.

  4. По клику на элемент показывать подробную информацию о нем в отдельном экране ("Название", "Город", "Улица" – много не надо, важен факт. Запросы данных о месте).

  5. Перегружать список и карту, если местоположение пользователя поменялось более чем на 100м.

  6. Поддержать поворот экрана (портретный и ландшафтный режим).

  7. Поддержать планшеты.

Комментарии по заданию:

  • Приложение НЕ нужно размещать в Play Store.

  • Исходный код должен быть написан на Java.

Результат:

  • Код проекта разместить на публичном репозитории.
